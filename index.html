Firebase Setup Required

This app needs Firebase config before it can connect to multiplayer services.

  1. Copy src/firebase-config.example.js to src/firebase-config.js.
  2. Paste your Firebase Web App config values.
  3. Enable Anonymous Authentication in Firebase Auth.
  4. Create a Realtime Database and apply database.rules.json.
  5. Reload or redeploy after saving config.

If this is deployed on Vercel, commit src/firebase-config.js with your project values.
